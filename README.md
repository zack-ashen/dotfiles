# i3-dotfiles
Dotfiles for i3 setup. Screenshots are running on Arch Linux.

Distro: Arch Linux
WM: i3-gaps
Bar: Polybar
Terminal: Kitty
Shell: Oh-My-Zsh (Theme: Typewritten)
Terminal Color Scheme: Dracula
Wallpaper: Mountains-Fog
